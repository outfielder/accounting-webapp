Upload CSV Files

Please upload your CSV files in the following formats:

Orders File Requirements

The Orders file should contain the following columns in this exact order:

  • Order ID
  • Order Date
  • Invoice
  • Checkout Price
  • VAT Amount
  • Supplier
  • VAT == 1/6 Checkout Price

Cancellations File Requirements

The Cancellations file should contain the following columns in this exact order:

  • Order (from Orderline) - Order ID associated with the cancellation
  • Credit Note Number - Unique identifier for the credit note
  • Cancellation/Refund Date - Date of the cancellation or refund
  • Checkout Price (from Order) (from Orderline) - Original checkout price of the order
  • VAT Amount (from Order) (from Orderline) - VAT amount of the cancelled order
  • Supplier (from Order) (from Orderline) - Supplier associated with the cancellation
  • Order fully cancelled (from Order) (from Orderline) - Indicates if the entire order was cancelled (1 for yes, 0 for no)
  • VAT == 1/6 Checkout Price (from Order) (from Orderline) - Indicates if VAT is one-sixth of the checkout price
